
########
dynamics
########

Molecular dynamics using interpolate potential energy surfaces


Installation
------------

To install dynamics, do:

.. code-block:: console

  git clone https://github.com/resmol/dynamics.git
  cd dynamics
  pip install .


Run tests (including coverage) with:

.. code-block:: console

  pytest tests
